About This Project

The AI Powered Disease Prediction System is a demo application that uses artificial intelligence logic to provide an early indication of potential diseases based on symptoms entered by the user.

Key Features

  • Simple AI-based prediction using weighted scoring logic.
  • Interactive and responsive web interface.
  • Chatbot assistant for guidance and help.
  • Text-to-speech output for prediction results (demo feature).

Technologies Used

  • HTML, CSS, JavaScript
  • Front-end AI logic for prediction
  • Optional: Text-to-Speech using Web Speech API

Our goal is to demonstrate how AI can be integrated into user-friendly health applications while maintaining a clean and professional interface.
